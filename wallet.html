Allows users to see contact lists and details.

Allows users to create new contacts or modify existing ones.

Allows users to import contact data from external files or export it.

Allows users to categorize contacts with labels for organization and filtering.

Allows users to remove contacts from the system.

Allows users to set up or modify automated sequences of messages.

Allows users to remove automated message sequences from system.
